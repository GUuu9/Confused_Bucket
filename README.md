# 주식 포트폴리오 관리 서비스

본 프로젝트는 서버 없이 브라우저 환경에서 동작하는 주식 포트폴리오 관리 서비스입니다. MVVM(Model-View-ViewModel) 아키텍처와 DI(의존성 주입) 패턴을 적용하여 확장성과 유지보수성을 고려하여 설계되었습니다.

## 🚀 주요 기능

### 1. 포트폴리오 관리 (카드형 UI)
- **주식 항목 추가**: 'Add Stock' 카드를 클릭하여 새로운 주식 항목을 동적으로 생성합니다.
- **정보 입력**: 종목명, 구매 가격, 수량, 현재 가격, 목표 매수/매도 라인을 입력합니다.
- **추가 매수**: 'Edit' 버튼을 통해 추가 매수 정보를 입력하면 평단가와 총 수량이 자동으로 재계산됩니다.
- **삭제**: 항목 상단의 삭제 버튼(`square-x`)을 통해 데이터를 제거할 수 있습니다. (삭제 시 확인 알림 제공)

### 2. 실시간 계산 및 분석
- **투자 총액 계산**: 구매 가격과 수량을 기반으로 투자 총액이 실시간으로 계산됩니다.
- **수익률/수익금 분석**: 현재 가격 입력 시, 구매가 대비 수익률과 수익금이 즉시 계산됩니다.
    - 수익 발생 시: **적색** 표시
    - 손실 발생 시: **청색** 표시
- **종합 요약 대시보드**: 전체 포트폴리오의 총 투자 금액, 현재 평가 금액, 전체 수익률을 화면 상단에서 한눈에 파악할 수 있습니다.

### 3. 데이터 관리 및 시각화 (테이블형 UI)
- **종합 리스트**: 전체 데이터를 테이블 형태로 확인 가능합니다.
- **정렬**: 각 헤더 클릭 시 이름, 투자 금액, 현재 가격, 수익률 기준 오름차순/내림차순 정렬이 가능합니다.
- **접기/펼치기**: 리스트 영역을 접거나 펼쳐서 화면 공간을 최적화할 수 있습니다.

### 4. 데이터 영속성 (Import/Export/Auto-save)
- **자동 저장**: `localStorage`를 사용하여 브라우저 새로고침 후에도 데이터가 유지됩니다.
- **Export**: 현재 포트폴리오 데이터를 JSON 파일(`portfolio.json`)로 내보냅니다.
- **Import**: JSON 파일을 불러와 포트폴리오를 복원합니다.
- **Reset**: 모든 데이터를 즉시 초기화합니다.

## 🏗 아키텍처 및 기술 스택

- **패턴**: MVVM, DI (Renderer 전용 컨테이너 사용)
- **언어**: TypeScript
- **환경**: Vite 기반 웹 애플리케이션
- **UI/UX**: Vanilla CSS (Flexbox/Grid), Lucide Icons
- **데이터 레이어**:
    - `StorageRepository`: 브라우저 기반 자동 저장(`localStorage`)
    - `FileRepository`: 브라우저 기반 JSON 파일 입출력
    - `StockViewModel`: 데이터 연산 및 비즈니스 로직 처리
